# go-engine
go语言开发通用代码库

## 内容
#### 基本
* 颜色支持
* 非阻塞终端
* 日志
* 循环数组
* 跳表
* 有锁链表
* 线程池
* 内存池
* 自失效Map
* 同步rpc
* FIFO消息
#### 环境
* nodejs
* shell
* 系统监控
#### 算法
* 德州扑克算法
* 可靠帧控制
#### 基础模块
* 可靠UDP
* 消息打解包
#### 功能模块
* DHT爬虫
* 网页爬虫
* ICMP隧道
* P2P网络
* HTML生成
* ChartJS适配
#### 第三方
* IP查询

## 依赖
* 设置GOPATH ``export GOPATH=$HOME/go``
* 安装nodejs，设置NODE_PATH ``export NODE_PATH=$GOPATH/src/github.com/esrrhs/go-engine/node/linux/node_modules/``
